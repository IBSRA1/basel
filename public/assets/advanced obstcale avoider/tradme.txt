================================================================================
                        DETAILED WIRING CONNECTIONS
================================================================================

STEP-BY-STEP WIRING INSTRUCTIONS:

POWER CONNECTIONS:
------------------
1. Arduino 5V Pin    ───→  HC-SR04 VCC Pin
2. Arduino 5V Pin    ───→  LCD Module VCC Pin
3. Arduino GND Pin   ───→  HC-SR04 GND Pin
4. Arduino GND Pin   ───→  LCD Module GND Pin

   NOTE: You may need to use a breadboard to split power rails if connecting
         multiple devices to the same 5V and GND pins.

HC-SR04 SENSOR CONNECTIONS:
---------------------------
5. Arduino Pin 9     ───→  HC-SR04 TRIG Pin (Trigger Signal)
6. Arduino Pin 10    ───→  HC-SR04 ECHO Pin (Echo Signal)

I2C LCD DISPLAY CONNECTIONS:
----------------------------
7. Arduino Pin A4    ───→  LCD SDA Pin (I2C Data)
8. Arduino Pin A5    ───→  LCD SCL Pin (I2C Clock)

   NOTE: On some Arduino boards, dedicated SDA/SCL pins are available
         separate from A4/A5. Either connection method works.

================================================================================
                        BREADBOARD LAYOUT (OPTIONAL)
================================================================================

If using a breadboard for neater connections:

1. Place Arduino Uno beside breadboard
2. Connect Arduino 5V to breadboard positive (+) rail (Red)
3. Connect Arduino GND to breadboard negative (-) rail (Black/Blue)
4. Place HC-SR04 sensor on breadboard
5. Mount I2C LCD on breadboard or keep separate

BREADBOARD POWER RAILS:
   Positive Rail (+) ─── Arduino 5V  ─┬─ HC-SR04 VCC
                                      └─ LCD VCC

   Negative Rail (-) ─── Arduino GND ─┬─ HC-SR04 GND
                                      └─ LCD GND

================================================================================
                    IMPORTANT NOTES AND PRECAUTIONS
================================================================================

⚠ VOLTAGE WARNING:
   - HC-SR04 operates at 5V. DO NOT connect to 3.3V systems directly.
   - Ensure all components share common ground (GND).

⚠ I2C ADDRESS:
   - Default I2C LCD address is usually 0x27 or 0x3F
   - Run I2C scanner sketch if LCD doesn't display
   - Adjust address in code if needed: LiquidCrystal_I2C lcd(0x27, 16, 2);

⚠ WIRING CHECKS:
   - Double-check all connections before powering on
   - Ensure no short circuits between VCC and GND
   - Verify correct pin numbers (D9, D10, A4, A5)

⚠ SENSOR PLACEMENT:
   - Mount HC-SR04 facing the target object
   - Keep sensor level and stable for accurate readings
   - Avoid soft materials (absorb sound) and extreme angles
   - Optimal range: 2 cm to 400 cm

⚠ LCD CONTRAST:
   - If LCD shows no text, adjust contrast potentiometer on I2C module
   - Usually located on back of LCD module (small blue screw)
   - Turn clockwise/counterclockwise until text is visible

⚠ POWER SUPPLY:
   - USB power (5V) is sufficient for this project
   - If using external power: 7-12V DC to Arduino VIN pin
   - Total current draw: ~100mA (well within USB limits)

================================================================================
                        TESTING PROCEDURE
================================================================================

1. VISUAL INSPECTION:
   □ Check all wire connections match diagram
   □ Verify no loose connections
   □ Ensure correct polarity (VCC to 5V, GND to GND)

2. POWER ON TEST:
   □ Connect Arduino to computer via USB
   □ LCD backlight should illuminate
   □ No smoke or burning smell (if present, disconnect immediately)

3. UPLOAD CODE:
   □ Upload the distance measurement code to Arduino
   □ Open Serial Monitor (9600 baud rate)

4. FUNCTIONAL TEST:
   □ LCD should show welcome message
   □ Distance readings should appear on LCD
   □ Move hand in front of sensor (2-400 cm range)
   □ Readings should change accordingly
   □ Serial Monitor should display same values

5. TROUBLESHOOTING:
   □ No LCD display: Check I2C address, adjust contrast
   □ Wrong readings: Check TRIG/ECHO connections
   □ No readings: Verify sensor VCC and GND connections
